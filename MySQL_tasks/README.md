### Директрия содержит файлы:

   * Human_Friends.sql

      Содержит листинг команд (запросов) в формате удобном для загрузки в графические клиенты для работы с сервером MySQL: 
      - MySQL Workbench
      - PhpMyAdmin

   * Human_Friends.txt

      Содержит листинг команд (запросов) в формате текстового файла
